XUL Interactor 1.0
date : 2007/02/08
Domain : Graphical User Interfaces
Description :  This metamodel is inspired by the XUL language but it is generic to all WIMP widget based User Interfaces. It describes a User Interfaces in terms of Interactors (e.g., button, windows, icons, list, etc.). It can be easily user for targetting any web interactor-based languages such as XUL, HTML, XAML, etc.
